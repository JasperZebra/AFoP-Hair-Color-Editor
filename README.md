# AFOP Hair Color Editor
Edit player hair color files (`.blueitemtype`) for Avatar: Frontiers of Pandora.

## How to Use

1. **Load files** — click **Load Folder** and browse to your hair color folder, or use **Load Files** to pick individual files.

2. **Edit a color** — click any color swatch to open the color picker.
   - Drag the **gradient square** to set saturation/brightness.
   - Drag the **rainbow bar** to shift hue.
   - Type a **hex code** directly or adjust the **R G B** spinboxes.

3. **Save** — click **Save Group** on a card to save all variants in that group at once.

> Files are grouped by number and shown with their region variants: **Base / Rainforest / Temperate / Veldt**. Each variant has 4 color slots. Only the hex color values are changed — everything else is left byte-for-byte identical.
